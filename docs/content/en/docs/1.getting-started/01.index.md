## What is Movk Nuxt Docs?

Movk Nuxt Docs is a Nuxt Layer built on top of the [UI documentation template](https://docs-template.nuxt.dev/), providing a complete documentation site experience. Powered by [Nuxt Content](https://content.nuxt.com), it supports Markdown and [MDC syntax](https://content.nuxt.com/docs/files/markdown#mdc-syntax) so you can focus on writing content.

## Key features

- Built on [Nuxt 4](https://nuxt.com) and [Nuxt UI](https://ui.nuxt.com)
- Supports [MDC syntax](https://content.nuxt.com/docs/files/markdown#mdc-syntax) (Markdown + Vue components)
- [Mermaid](https://mermaid.js.org/intro/) diagram support (optional, for flowcharts, sequence diagrams, class diagrams, and more)
- Automated component documentation (Props, Slots, Emits, examples)
- Git integration (commit history, edit links)
- Automatic sidebar navigation and full-text search
- Dark mode support
- Responsive design (mobile first)
- SEO optimization and OG image generation
- Full TypeScript type support
- Opt-in multilingual support based on `@nuxtjs/i18n`, with per-language content collections, a built-in language switcher, and hreflang SEO
- AI integrations (LLMs.txt, MCP Server, AI Chat, Skills)
