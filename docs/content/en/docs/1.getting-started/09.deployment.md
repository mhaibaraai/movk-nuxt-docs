## Build for production

```bash
pnpm build
```

The build output is located in the `.output/` directory and includes both server-side and client-side code.

## Vercel deployment

### Automatic deployment

1. Create a new project on Vercel
2. Connect your Git repository
3. Configure the environment variables:
   ```bash
   GITHUB_TOKEN=your-token
   AI_GATEWAY_API_KEY=your-key
   ```
4. The deploy command defaults to `nuxt build`

## Environment variables

| Variable | Purpose | Required when |
|------|------|---------|
| `GITHUB_TOKEN`{lang="ts-type"} | Git integration | Displaying commit history |
| `AI_GATEWAY_API_KEY`{lang="ts-type"} | AI Chat feature | Using AI Chat |
