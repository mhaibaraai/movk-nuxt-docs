## 构建生产版本

```bash
pnpm build
```

构建产物位于 `.output/` 目录，包含服务端和客户端代码。

## Vercel 部署

### 自动部署

1. 在 Vercel 创建新项目
2. 连接 Git 仓库
3. 配置环境变量：
   ```bash
   GITHUB_TOKEN=your-token
   AI_GATEWAY_API_KEY=your-key
   ```
4. 部署命令默认为 `nuxt build`

## 环境变量

| 变量 | 用途 | 必需场景 |
|------|------|---------|
| `GITHUB_TOKEN`{lang="ts-type"} | Git 集成 | 显示提交历史 |
| `AI_GATEWAY_API_KEY`{lang="ts-type"} | AI Chat 功能 | 使用 AI Chat |
