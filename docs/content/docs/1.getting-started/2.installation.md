## 快速入门

您可以通过以下命令快速启动一个全新的项目：

```bash [Terminal]
npx nuxi init -t gh:mhaibaraai/movk-nuxt-docs/template my-docs
cd my-docs
pnpm run dev
```

项目启动后，您可以在 <http://localhost:3000> 访问本地预览。

## 层集成

Movk Nuxt Docs 采用 **Nuxt 层架构**。您可以在 `nuxt.config.ts` 文件中通过 `extends: ['@movk/nuxt-docs']` 来直接继承 Movk Nuxt Docs 层：

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['@movk/nuxt-docs']
})
```
