## 用法

在您的 Markdown 文档中，使用以下语法来生成不同类型的 API 文档：

::code-preview

:::div{class="flex flex-col w-full"}
### Props
:component-props{slug=Api}

### Slots
:component-slots{slug=Api}

### Emits
:component-emits{slug=Api}
:::
