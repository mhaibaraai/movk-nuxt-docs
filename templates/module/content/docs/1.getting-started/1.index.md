## 什么是 Movk Nuxt Docs?

Movk Nuxt Docs 是一个基于 [UI 文档模板](https://docs-template.nuxt.dev/) 的 Nuxt Layer，提供完整的文档站点功能。通过 [Nuxt Content](https://content.nuxt.com) 支持 Markdown 和 [MDC 语法](https://content.nuxt.com/docs/files/markdown#mdc-syntax)，专注于内容创作。

## 主要特性

- 基于 [Nuxt 4](https://nuxt.com) 和 [Nuxt UI](https://ui.nuxt.com)
- 支持 [MDC 语法](https://content.nuxt.com/docs/files/markdown#mdc-syntax)（Markdown + Vue 组件）
- [Mermaid](https://mermaid.js.org/intro/) 图表支持（流程图、时序图、类图等可视化图表）
- 组件文档自动生成（Props、Slots、Emits、示例）
- Git 集成（提交历史、编辑链接）
- 自动侧边栏导航和全文搜索
- 暗黑模式支持
- 响应式设计（移动优先）
- SEO 优化和 OG 图片生成
- 完整 TypeScript 类型支持
- AI 功能集成（LLMs.txt、MCP Server、AI Chat）
