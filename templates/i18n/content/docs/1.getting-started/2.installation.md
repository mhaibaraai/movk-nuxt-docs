---

## 创建项目

```bash [Terminal]
npx nuxi init -t gh:mhaibaraai/movk-nuxt-docs/templates/i18n my-docs
cd my-docs
pnpm install
pnpm dev
```

启动后访问 <http://localhost:3000>（中文）或 <http://localhost:3000/en>（英文）。

## 添加新语言

1. 在 `nuxt.config.ts` 的 `i18n.locales` 中注册新语言：

```ts [nuxt.config.ts]
i18n: {
  defaultLocale: 'zh-CN',
  locales: [
    { code: 'zh-CN', name: '简体中文', file: 'zh-CN.json' },
    { code: 'en', name: 'English', file: 'en.json' },
    { code: 'ja', name: '日本語', file: 'ja.json' }
  ]
}
```

2. 新建对应的内容目录 `content/ja/`，否则该语言会在构建期被自动过滤并告警。

> 界面文案翻译文件由 `@movk/nuxt-docs` 内置；当前内置 `zh-CN`、`en`，其余语言需自行提供。
