## 多语言文档模板

这是基于 [Movk Nuxt Docs](https://docs.mhaibaraai.cn) 的多语言站点模板，预置 `zh-CN`（默认）与 `en` 两种语言，已通过 [@nuxtjs/i18n](https://i18n.nuxtjs.org) 完成配置。

- **默认语言（简体中文）** 内容位于 `content/` 根目录，路由无前缀（如 `/docs/getting-started`）。
- **英文** 内容位于 `content/en/`，路由带前缀（如 `/en/docs/getting-started`）。

## 目录结构

```bash
content/
├── index.md                      # 中文首页
├── docs/                         # 中文文档 → /docs/*
│   └── 1.getting-started/
│       ├── 1.index.md
│       └── 2.installation.md
└── en/                           # 英文内容
    ├── index.md                  # 英文首页 → /en
    └── docs/                     # 英文文档 → /en/docs/*
        └── 1.getting-started/
            ├── 1.index.md
            └── 2.installation.md
```

## 下一步

- 在 `content/{locale}/docs/` 下新增 Markdown 文件即可自动生成对应语言的路由。
- 如需新增语言，在 `nuxt.config.ts` 的 `i18n.locales` 注册，并补充对应的 `content/{code}/` 目录。
- 更多用法见 [国际化文档](https://docs.mhaibaraai.cn/docs/i18n)。
