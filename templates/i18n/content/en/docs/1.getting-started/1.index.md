## Multilingual docs template

This is a multilingual site template built on [Movk Nuxt Docs](https://docs.mhaibaraai.cn), preconfigured with `zh-CN` (default) and `en` via [@nuxtjs/i18n](https://i18n.nuxtjs.org).

- **Default locale (Simplified Chinese)** content lives at the `content/` root with no route prefix (e.g. `/docs/getting-started`).
- **English** content lives under `content/en/` with a route prefix (e.g. `/en/docs/getting-started`).

## Directory structure

```bash
content/
├── index.md                      # Chinese home
├── docs/                         # Chinese docs → /docs/*
│   └── 1.getting-started/
│       ├── 1.index.md
│       └── 2.installation.md
└── en/                           # English content
    ├── index.md                  # English home → /en
    └── docs/                     # English docs → /en/docs/*
        └── 1.getting-started/
            ├── 1.index.md
            └── 2.installation.md
```

## Next steps

- Add Markdown files under `content/{locale}/docs/` to generate routes for that locale automatically.
- To add a language, register it in `i18n.locales` in `nuxt.config.ts` and create the matching `content/{code}/` directory.
- See the [internationalization guide](https://docs.mhaibaraai.cn/en/docs/i18n) for more.
