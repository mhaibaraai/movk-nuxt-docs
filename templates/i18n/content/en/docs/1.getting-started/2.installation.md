title: Installation
description: Create and run the multilingual documentation site template.
---

## Create the project

```bash [Terminal]
npx nuxi init -t gh:mhaibaraai/movk-nuxt-docs/templates/i18n my-docs
cd my-docs
pnpm install
pnpm dev
```

Then open <http://localhost:3000> (Chinese) or <http://localhost:3000/en> (English).

## Add a new language

1. Register the locale in `i18n.locales` in `nuxt.config.ts`:

```ts [nuxt.config.ts]
i18n: {
  defaultLocale: 'zh-CN',
  locales: [
    { code: 'zh-CN', name: '简体中文', file: 'zh-CN.json' },
    { code: 'en', name: 'English', file: 'en.json' },
    { code: 'ja', name: '日本語', file: 'ja.json' }
  ]
}
```

2. Create the matching content directory `content/ja/`, otherwise the locale is filtered out at build time with a warning.

> UI translation files are built into `@movk/nuxt-docs`; `zh-CN` and `en` are included, other locales must be provided yourself.
