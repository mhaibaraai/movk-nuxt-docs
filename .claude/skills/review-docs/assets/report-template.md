### 技术：MDC 组件语法错误

#### [页面路径]：组件嵌套层级错误

**文件：** `content/docs/[path]/[page].md:行号`

**问题：** 嵌套层级跳跃（从 `::` 直接到 `::::`）

**当前：**
```mdc
::card-group
  ::::card{icon="i-lucide-book"}
  内容
  ::::
::
```

**应为：**
```mdc
::card-group
  :::card{icon="i-lucide-book"}
  内容
  :::
::
```

**影响：** 组件渲染错误，页面显示异常
